# Projeto Fullstack + Mobile

Este repositório contém três projetos integrados:

- Backend: Java + Spring Boot (maven)
- Frontend: React 19 + TypeScript + Mantine
- Mobile App: Flutter


## 1. Ferramentas e versões instaladas

- **Java**: `java 17.0.10 2024-01-16 LTS` 
- **Node.js**: `v18.20.8`  
- **Flutter**: `3.38.3 channel stable `  
- **Maven**: `Apache Maven 3.9.11`

## 2. Portas Utilizadas
  Backend: 8080
  Frontend: 5173
  Mobile: 49913


## 3. Como executar cada projeto

### Backend (Java + Spring Boot)
cd backend
mvn clean install
mvn spring-boot:run
Porta: 8080 (padrão)

### Frontend 
cd frontend
npm install
npm run dev

### Mobile
cd mobile_app
flutter run

3. IDEs/Editores usados

Backend: IntelliJ IDEA community

Frontend: VS Code

Mobile App: Android Studio / VS Code (Flutter)
